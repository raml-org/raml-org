The simplest way to design APIs

RESTful API Modeling Language (RAML) makes it easy to manage the whole API lifecycle from design to sharing. It's concise -- you only write what you need to define -- and reusable. It's interoperable: it works with OpenAPI (OAS).

Write once. Use many. Creative laziness encouraged.
